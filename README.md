# Telegram Bot

Простой многофункциональный Telegram бот на Python с использованием библиотеки pyTelegramBotAPI.

## Функциональность

Бот поддерживает следующие команды:

- `/start` или `/help` - показать справку по командам
- `/joke` - случайная шутка
- `/quote` - мудрая цитата  
- `/fact` - интересный факт
- `/weather` - текущая погода в Улан-Удэ
- `/coin` - подбросить монетку
- `/car цвет марка [модель]` - создать виртуальный автомобиль

Структура проекта

    ├── bot.py          # Основной файл бота с обработчиками команд
    ├── config.py       # Конфигурация с токеном бота
    ├── car.py          # Класс Car для создания автомобилей
    ├── lists.py        # Списки шуток, цитат и фактов
    ├── weather.py      # Модуль для работы с погодой (OpenWeatherMap API)
    └── README.md       # Этот файл

Используемые библиотеки

    pyTelegramBotAPI - для работы с Telegram Bot API

    requests - для HTTP-запросов к API погоды

    random - для генерации случайных значений

Примеры использования
Создание автомобиля

/car красный Tesla Model S

Получение погоды

/weather

Случайная шутка

/joke

Особенности реализации

    Модульная структура с разделением функционала по файлам

    Обработка ошибок в командах

    Использование OpenWeatherMap API для получения реальных данных о погоде

    Объектно-ориентированный подход с классом Car
